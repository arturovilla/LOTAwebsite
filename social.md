## LOTA Update — Docs, Custom TouchDesigner Component, ARKit Tracking & More

Big thank you to everyone who's been beta testing and those who've signed up for email updates — your feedback has been shaping this project in a real way. Here's what's new:

**Full Documentation**
The site now has a complete docs page — capture modes, streaming setup, TouchDesigner integration, 3D export pipelines, and accessibility. Everything you need to get started.

**LOTAPoints.tox — Drop-in TouchDesigner Component**
We built a custom .tox that handles live PLY point cloud streaming out of the box. Drag it into your project, it defaults to port 9848, and it just works. No scripting required. Available for download on the docs page.

**ARKit Body & Face Tracking (Coming Soon)**
Real-time body tracking (91 joints) and face capture (52 blend shapes) streamed over OSC. The site now showcases this upcoming feature — it's currently pending Apple review and will be available to beta testers soon.

**Changelog**
Every update is now tracked on a dedicated changelog page so you can follow development as it happens.
